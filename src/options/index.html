Setup Guide

  1. Install Anki desktop app from ankiweb.net
  2. Install AnkiConnect add-on (code: 2055492159)
  3. Restart Anki
  4. Click "Create Anki Model" above to set up the card template
  5. Select your preferred deck
  6. Visit Kagi Translate and start creating flashcards!
